    The What I.F. Machine

    Lucy Oliverio, Anthony Catricala, Dakota Jones, Patrick Modica, Michael Johnson


    The What I.F. Machine is a scriptable text-based adventure game engine. It is written in python and it uses JSON files to store information. My teammates and I worked on it for our software engineering class. I personally worked on coding the parsers. This means I was in charge of manipulating the data extracted from the stored files. Please refer to "The Rundown" for more information. Please note: I did not work on the code associated with user scripts or triggers in parsers. I also did not work on the "What “IF” Machine User Manual" provided below.
